Descrição do Projeto:
Lanchonete Virtual com Cadastro de Produtos e Pedidos

Este projeto se trata de uma lanchonete virtual que permite a gestão de produtos e pedidos. A aplicação realiza operações como:
Cadastro de Produtos: Permite adicionar novos produtos ao cardápio.
Cadastro de Pedidos: Realiza o cadastro de pedidos realizados pelos clientes, com uma lista dos produtos comprados.
Operações de Listar Produtos e Pedidos: Permite visualizar todos os produtos cadastrados e os pedidos realizados.
Adicionar, Remover e Visualizar Produtos: O administrador pode adicionar novos produtos, remover produtos do cardápio e visualizar todos os produtos cadastrados.

Passo a passo de como executar o código:
1. abra Visual Studio 2022
2. open a local folder
3. selecione a pasta "cp1-"
4. rode o codigo com o debug


Versão do .NET/C# utilizada:
A aplicação utiliza C# e .NET 8.0.
